☁️ GITHUB SYNC
Your data lives in your GitHub repo — no ad blockers, no third parties, works everywhere
HOW TO SET UP (3 minutes, free)
  1. Go to github.com → Settings → Developer Settings → Personal Access Tokens → Tokens (classic)
  2. Click Generate new token (classic)
  3. Give it a name, set expiry, and tick only the repo checkbox
  4. Copy the token (starts with ghp_...)
  5. Fill in the fields below and click CONNECT & PUSH DATA
Stored only on your device. Never sent anywhere except GitHub.
